## 🚨 발견된 문제점

### 1️⃣ **자동 수집 크론이 없다** (가장 심각)
- **문제**: 발송만 자동 (월·수·금 9시), 수집은 수동 또는 없음
- **결과**: 수집 데이터가 없으면 발송도 0건으로 됨
- **예시**: 7/14(수) 미발송 (수집 크론 없어서)
- **해결책**: Vercel Cron에 수집 크론 추가 필요
  ```json
  // vercel.json에 추가해야 함
  {
    "path": "/api/cron/collect-articles",
    "schedule": "0 23 * * 0,2,4"  // 월·수·금 8시 (발송 1시간 전)
  }
  ```

### 2️⃣ **발송 크론이 두 군데** (중복 위험)
- **Vercel Cron**: `vercel.json` → `0 0 * * 1,3,5`
- **GitHub Actions**: `.github/workflows/daily-digest-send.yml` → `0 0 * * 1,3,5`
- **위험**: 오전 9시에 메일이 두 번 갈 수 있음
- **해결책**: 하나만 남기기 (권장: Vercel Cron, GitHub Actions 제거)

### 3️⃣ **DB 인덱스 에러 발생 과거** (현재 해결됨)
- **당시**: 7/10 `Article.link` 컬럼이 너무 길어서 인덱스 오류
- **현황**: 지금은 정상 작동 (자동 수정되거나 누가 처리함)
